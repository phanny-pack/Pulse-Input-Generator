Pulse Input Generator

Input desired pulse width, gap width (width between the two pulses in a set), and pulse set width (width between each set of pulses):

clocks* μs μs

Made by Chris Phan -- August, 2020

*Note* - each Arduino Uno clock is 62.5 nanoseconds - 2 gives 125 ns on top of the 150 ns it takes to execute digitalWriteFast. EDIT: incorporating serial inputs adds ~550 ns to pulse width, so entering 0.5 gives ~750 ns pulse width: the smallest I could get it so far
